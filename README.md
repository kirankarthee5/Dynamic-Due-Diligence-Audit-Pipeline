# Dynamic-Due-Diligence-Audit-Pipeline
Python &amp; SQL ETL pipeline for my Capstone on Dynamic Due Diligence. Uses NumPy to simulate 100k stochastic retail trades, SQLite3 for relational storage, and Pandas/Seaborn to calculate risk exposure. Replaces manual Excel audits with an automated architecture that processes 100k rows in &lt;2 seconds for instant compliance insights.
